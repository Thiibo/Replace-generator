    Use \n to write a new line.
    Use \i to write the the current iteration number.
    Use \c to write the closed counter value. This value increases every time '\c' is called and will reset every new iteration.
    Use \o to write the open counter value. This value increases every time '\o' is called and will NEVER reset.
    Use \\ to write '\' (thus escaping the special cases).
